Tabs

Description:

This project will enable people to take control of their clutter, organize it, and keep tabs on it as they retrieve and use it. To that end, the Personal Inventory system will digitize the user’s space into containers and sub-containers, load the items they store in each space into the system, and then allow the user to query those items from the web to either use the item or for informational purposes.
